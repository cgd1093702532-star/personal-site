# 本地数据库

M1 阶段使用 **SQLite + 本地 REST API**，让预览页和小程序创建的数据可持久化、可读取、可在页面间流转。

## 架构

```
预览页 / 小程序
    ↓  HTTP（优先）
本地 API（8787）  server/local_api.py
    ↓
SQLite  data/local.db
    ↑ 首次启动
种子数据  data/seed.json  ← scripts/generate-seed.py

小程序 API 不可用时
    ↓
wx.storage 镜像  miniprogram/utils/store.js
```

## 快速开始

```bash
# 1. 生成种子（从 mock.js + heroes-data.js）
python3 scripts/generate-seed.py

# 2. 启动本地 API
bash scripts/start-local-db.sh

# 3. 启动预览（另开终端）
bash scripts/start-preview.sh

# 或一键启动 API + 预览
bash scripts/start-dev.sh
```

## 主要 API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/heroes` | 英雄列表 |
| GET | `/api/heroes/:id` | 英雄详情 |
| PUT | `/api/heroes/:id` | 更新英雄资料（关于我、荣誉等） |
| GET | `/api/recruitments/mine/:tab` | 我的招募（active/ended/draft） |
| POST | `/api/recruitments` | 新建招募 |
| PUT | `/api/recruitments/:id` | 更新招募 |
| DELETE | `/api/recruitments/:id` | 删除招募 |
| GET/PUT | `/api/app-state/:key` | 应用状态（报名、申请等） |
| POST | `/api/heroes/apply` | 提交英雄认证申请 |
| GET | `/api/heroes/apply/status` | 查询当前用户申请状态 |
| POST | `/api/heroes/apply/withdraw` | 撤回待审核申请（预览用） |
| GET | `/api/admin/applications` | 后台：申请列表（`?status=pending`） |
| GET | `/api/admin/applications/:id` | 后台：申请详情 |
| POST | `/api/admin/applications/:id/approve` | 后台：批准申请 |
| POST | `/api/admin/applications/:id/reject` | 后台：驳回申请 |
| POST | `/api/reset` | 重置为种子数据（开发用） |

## 已持久化的保存点

| 页面/功能 | 存储位置 | 说明 |
|-----------|----------|------|
| 英雄资料编辑 | `heroes` 表 | 关于我、荣誉、精彩瞬间、证书 |
| 发布/编辑招募 | `recruitments` 表 | 草稿、发布、下架、删除 |
| 招募详情报名 | `app_state.my_signups` + 招募 `signed` | 报名记录与人数 |
| 报名名单确认 | `app_state.my_signups` | 状态更新 |
| 申请成为英雄 | `hero_applications` 表 + `app_state` 同步 | 表单提交、后台审核 |
| 发布课程 | `app_state.my_courses` | 课程创建记录 |
| 课程报名 | `app_state.my_signups` | 课程类报名 |
| 证书上传（申请） | `app_state.hero_apply_certs` | 证书图片路径 |
| 个人中心英雄状态 | `hero_applications` / `app_state.mock_hero_role` | 认证/审核状态 |

预览页通过 `preview/assets/db-client.js`（`HeroPlazaDB`）写入同一套 API。

## 代码入口

| 端 | 文件 |
|----|------|
| 服务端 | `server/db.py`、`server/local_api.py` |
| 预览客户端 | `preview/assets/db-client.js` |

## 小程序联调

微信开发者工具 → 详情 → 本地设置 → **不校验合法域名**，即可访问 `http://127.0.0.1:8787`。

## 重置数据

```bash
curl -X POST http://127.0.0.1:8787/api/reset
```

或删除 `data/local.db` 后重启 API。小程序离线镜像可清除 `wx.storage` 键 `hero-plaza-local-db`。
