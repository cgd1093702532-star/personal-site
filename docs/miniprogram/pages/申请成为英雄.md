## 1. 页面概述

| 项 | 值 |
|---|---|
| 路由 | `miniprogram/pages/hero-apply/hero-apply` |
| 导航标题 | 申请成为英雄 |
| 入口 | 个人中心「申请成为英雄」 |

## 2. 页面结构

1. **页头引导** — 认证成为英雄教练 + 副标题
2. **基本信息** — 姓名、手机号（可更换）、项目类型、常驻城市
3. **资质认证** — 教练资质等级、从业年限、资质证书上传
4. **荣誉与成就** — 曾获荣誉（选填）
5. **个人简介** — 详细介绍、个人展示视频（选填）
6. **协议勾选 + 提交**

## 3. 表单字段

| 字段 | 类型 | 必填 | 校验 / 说明 |
|------|------|------|------|
| 姓名 | 文本 | 是 | 2–20 字符 |
| 手机号 | 文本 | 是 | 11 位；展示脱敏，支持更换 |
| 项目类型 | 多选标签 | 是 | ≥1 项：帆船 / 皮划艇 / 桨板 / 潜水 |
| 常驻城市 | 省市区选择 | 是 | region picker |
| 教练资质等级 | 单选 | 是 | 国家级教练 / 省级教练 / ACA认证 / ISA认证 / 其他 |
| 从业年限 | 单选标签 | 否 | 1-3年 / 3-5年 / 5-10年 / 10年+ |
| 资质证书 | 图片 | 是 | 最多 5 张，网格展示，可继续添加 |
| 曾获荣誉 | 多行 | 否 | 最多 200 字 |
| 详细介绍 | 多行 | 是 | 10–500 字 |
| 个人展示视频 | 视频 | 否 | 建议 30 秒以内 |
| 协议勾选 | 复选 | 是 | 英雄认证协议等 |

## 5. 业务流程

1. 填写 → 前端校验（含协议勾选）
2. `POST /api/heroes/apply`
3. 成功 → [认证成功](./认证成功.md)

## 6. 业务规则

- 已有 `pending` 申请：不可重复提交
- 已 `approved`：不可进入本页
- `rejected`：可重新提交
