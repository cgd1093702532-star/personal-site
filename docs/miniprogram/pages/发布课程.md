## 2. 业务需求

### 2.1 业务目标

- 快速发布课程：Banner、名称、价格、人数
- **课程详情**不在小程序编辑，提示前往管理后台富文本
- 发布成功 → [我的课程](./我的课程.md)

### 2.2 适用角色与权限

| 角色 | 可否进入 |
|------|----------|
| 已认证英雄 | ✅ |
| 其他 | ❌ |

### 2.3 核心业务规则

1. Banner：可多张 swiper 展示；「+ 添加 Banner 图片」每次 chooseMedia 1 张追加
2. 无 Banner 时显示占位 **点击上传 Banner 图片**
3. 提交校验：title≥2 字；price≥0；headcount≥1
4. `addMyCourse` 默认 `location:'待完善'`, `timeDisplay:'待排期'`
5. 成功 redirect my-courses 800ms 延迟

## 4. 字段与校验矩阵

| 字段 key | 标签 | 控件 | 必填 | 规则 | 错误 Toast | API 字段 |
|----------|------|------|------|------|------------|----------|
| title | 课程名称 | input | ✅ | trim 长度 ≥2 | **请填写课程名称** | title |
| price | 课程价格 | input digit | ✅ | Number≥0 | **请填写课程价格** | price |
| headcount | 课程人数 | input number | ✅ | Number≥1 | **请填写课程人数** | total/headcount |
| banners | Banner | image[] | ❌ | chooseMedia | — | banner_images |

### 4.1 提交 payload

```json
{
  "title": "周末帆船体验课",
  "price": 680,
  "total": 12,
  "headcount": 12,
  "banner_images": ["wxfile://..."],
  "location": "待完善",
  "timeDisplay": "待排期",
  "listTab": "active",
  "signed": 0
}
```

### 4.2 提交前校验顺序

1. title.trim().length ≥ 2  
2. price 非空且 Number(price) ≥ 0  
3. headcount 存在且 Number(headcount) ≥ 1
