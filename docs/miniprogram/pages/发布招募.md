# 发布赛事招募

> 状态：已实现 · P0 · M2

---

## 1. 页面概述

| 项 | 值 |
|---|---|
| 路由 | `pages/recruitment-create/recruitment-create` |
| 入口 | 个人中心 · 已认证 ·「发布招募/课程」→ 发布赛事招募 |
| 权限 | 仅 `heroes/me.status=approved` |

## 2. 表单分区

### 基本信息

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 招募标题 | 文本 | 是 | 2–50 字符 |
| 活动地点 | 文本 + 地图选点 | 是 | 支持 `chooseLocation` |
| 开始日期 | 日期 | 是 | ≥ 今天 |
| 开始时间 | 时间 | 否 | 默认 09:00 |
| 结束日期 | 日期 | 否 | — |
| 结束时间 | 时间 | 否 | 默认 17:00 |

### 招募设置

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 招募人数 | 数字 | 是 | 1–100 |
| 费用（元/人） | 数字 | 是 | 0–99999，0=免费 |
| 报名截止时间 | 日期 | 是 | 须早于活动开始日期 |
| 招募对象 | 多选标签 | 否 | 青少年 / 成人 / 亲子 / 新手 |

### 活动详情

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 活动亮点 | 多行文本 | 否 | 最多 500 字 |
| 详细描述 | 多行文本 | 是 | 10–2000 字 |
| 封面图片 | 图片 | 否 | 建议 16:9 |

### 联系方式

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 咨询手机 | 手机号 | 是 | 11 位 |

## 3. 底部操作

| 按钮 | 行为 |
|------|------|
| 保存草稿 | 本地存储草稿（`recruitment-event-draft`） |
| 立即发布 | 校验 → `POST /api/recruitments` → [我的招募](./我的招募.md) |
