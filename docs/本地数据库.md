# 本地数据库

本地 API（SQLite）**仅服务认证治理**；小程序业务数据走 `store.js` / `mock.js`。

## 架构

```
管理后台（认证）
    ↓  HTTP
本地 API（8787）  server/local_api.py
    ↓
SQLite  data/local.db
    ↑ 首次启动
种子数据  data/seed.json

小程序 · 认证（申请/状态/资料变更）
    ↓  HTTP（api.js）
本地 API 同上

小程序 · 业务（英雄/招募/课程/报名/评价等）
    ↓
wx.storage  miniprogram/utils/store.js
    ↑ 种子
mock.js + data.js 内 DEFAULT_*
```

## 快速开始

```bash
# 1. 生成种子（供方/申请等认证相关种子）
python3 scripts/generate-seed.py

# 2. 启动本地 API
bash scripts/start-local-db.sh

# 3. 启动预览（另开终端）
bash scripts/start-preview.sh

# 或一键启动 API + 预览
bash scripts/start-dev.sh
```

## 保留的 API（认证）

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/health` | 健康检查 |
| GET | `/api/heroes/apply/status` | 申请状态 |
| POST | `/api/heroes/apply` | 提交申请 |
| POST | `/api/heroes/apply/withdraw` | 撤回申请 |
| POST | `/api/heroes/:id/profile-changes` | 提交资料变更 |
| POST | `/api/heroes/:id/profile-changes/withdraw` | 撤回资料变更 |
| GET/PUT | `/api/app-state/:key` | 应用状态（小程序仅同步 `mock_hero_role`） |
| GET | `/api/admin/applications*` | 申请列表/详情 |
| POST | `/api/admin/applications/:id/approve\|reject` | 审核申请 |
| DELETE | `/api/admin/applications/:id` | 删除申请 |
| GET/POST | `/api/admin/heroes*` | 供方列表/创建/启停/删除 |
| GET | `/api/admin/users` | 用户列表 |
| POST | `/api/admin/users/:id/disable\|enable` | 用户启停 |
| GET | `/api/admin/profile-changes*` | 主页变更列表/详情 |
| POST | `/api/admin/profile-changes/:id/approve\|reject` | 审核变更 |
| POST | `/api/reset` | 重置种子（开发用） |

## 已移除的业务 API

`/api/heroes` 列表/详情、`/api/recruitments*`、`/api/courses*`、`/api/signups*`、`/api/reviews*`、`/api/admin/dashboard|recruitments|courses|signups|reviews|settings` 等 — 小程序改走本地 mock。

## 产品约定

- 审核通过只更新当前用户认证状态；**不**同步刷新广场英雄 mock 列表
- 详见 `docs/superpowers/specs/2026-07-14-admin-auth-only-mock-design.md`
