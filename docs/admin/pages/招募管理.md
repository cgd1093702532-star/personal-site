# 招募管理（管理后台）

> 状态：草稿 · P0 · M1

---

## 1. 列表

- **Tab：** 全部 / 招募中 / 已满员 / 已结束 / 已取消
- **筛选：** 场景、类型、发布时间
- **搜索：** 标题
- **表格：** 标题、场景、英雄、报名情况、价格、时间、状态
- **操作：** 查看详情、关闭招募

## 2. 详情弹窗

- 基本信息、时间地点、进度、富文本详情
- 已报名人员：姓名、电话、时间、支付/报名状态
- **关闭：** `POST /api/admin/recruitments/{recruit_id}/close`
- **编辑招募** — 字段同小程序发布表单

## 3. API

- `GET /api/admin/recruitments`
- `GET /api/admin/recruitments/{recruit_id}`
- `GET /api/admin/recruitments/{recruit_id}/signups`
