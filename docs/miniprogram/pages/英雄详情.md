# 英雄详情

> 状态：已确认 · P0 · M1  
> 最后更新：2026-07-06

---

## 1. 页面概述

| 项 | 值 |
|---|---|
| 路由 | `pages/hero-detail/hero-detail` |
| 导航 | 子页（从首页/英雄广场进入） |
| 参数 | `id`（hero_id） |

## 2. 页面结构

1. **个人主页头部** — 封面渐变 + 居中圆形头像 + 姓名 + 项目/年限 + 星级 + 标签 + 学员/评分/荣誉数据
2. **关于我** — 标题「关于我」，正文最多 300 字
3. **过往荣誉** — 小图标、荣誉名称、荣誉简介（最多 20 字）
4. **精彩瞬间** — 图片网格，最多 10 张，点击打开大图查看器
5. **资质证书** — 横向滑动展示，默认可见 3 张，含证书名称（如国家级教练证、救生员证、ACA证），点击打开大图查看器
6. **赛事招募** — 资质证书下方，显示招募数量；每条含名称、起止时间、赛事地点、费用（元/人）、报名进度
7. **底部留白** — 页面滑到底部时保留间距，避免内容贴底
8. **分享** — 封面区分享按钮，支持分享海报与分享小程序卡片到微信

## 3. 数据联动

| 详情页字段 | 英雄广场来源 |
|-----------|-------------|
| 3 个标签 | `honor_titles`（前 2 个）+ `cert_badges`（补足至 3 个） |
| 姓名 / 评分 | `name` / `rating` |
| 关于我 | `about_me`（独立字段，列表页 `bio` 为摘要） |
| 赛事招募 | 按 `hero_id` 关联招募列表，展示名称、起止时间、赛事地点、费用、报名进度 |

## 4. 功能说明

### 赛事招募列表

| 字段 | 说明 |
|------|------|
| 区块标题 | 「赛事招募」，右侧显示「共 N 个」 |
| 招募名称 | 如「周末帆船体验营」 |
| 时间 | `start_at` ~ `end_at`，同日显示「7月12日 09:30 - 16:00」 |
| 赛事地点 | 前缀「赛事地点：」+ 举办地点 |
| 费用 | `¥{fee}/人` |
| 报名 | 未满：`共招募 {total} 名，已报 {signed} 名`；已满：`已满 {total} 名，可继续报名`（满员后仍可报名） |

### 页面分享

| 能力 | 说明 |
|------|------|
| 入口 | 头像下方居中「分享」按钮 |
| 分享海报 | Canvas 生成教练名片海报，支持保存相册、分享给好友 |
| 分享小程序卡片 | `onShareAppMessage` + `open-type="share"`，分享带 `id` 的详情页路径 |

### 大图查看器（精彩瞬间 / 资质证书）

| 能力 | 说明 |
|------|------|
| 打开方式 | 点击缩略图，从当前图进入 |
| 计数 | 顶部显示「当前第几张 / 共几张」 |
| 切换 | 左右滑动（小程序 swiper）或左右箭头（预览） |
| 缩放 | 双指捏合放大缩小（小程序 movable-view；预览支持双指、滚轮、双击） |
| 保存 | 小程序写入相册；预览触发下载 |
| 分享 | 小程序调用分享图片菜单；预览使用系统分享或复制链接 |

### 其他操作

| 操作 | 行为 |
|------|------|
| 点击招募项 | → [招募详情](./招募详情.md) |
