## 1. 页面业务目标

「我的招募」是**已认证英雄**管理自己发布的赛事招募的列表页。

主要解决三件事：

1. **分状态查看**：草稿、招募、招募已结束三个 Tab，一眼看清每条招募处于什么阶段
2. **日常运营**：查看报名、编辑信息、下架活动；草稿可继续编辑或删除
3. **粗略经营数据**：非草稿 Tab 顶部展示本月报名人次与本月收入（当前为演示数字）

## 2. 登录和身份描述

| 身份 | 用户大概情况 | 页面上多出来 / 不一样的地方 |
|------|--------------|------------------------------|
| 已认证英雄 | 从个人中心「我的招募」或发布成功后进入 | 完整三 Tab 列表与操作按钮 |
| 未认证 / 普通用户 | 无入口 | 不应进入本页 |

> 从其他页面返回时，可直接落在「草稿」「招募」或「招募已结束」某一 Tab。

### 2.1 草稿 Tab

展示尚未发布的招募。卡片带 📝 图标与「未发布 · 信息待完善」提示；底部按钮为「继续编辑」「删除」。空态引导去「发布赛事招募」。

### 2.2 招募 Tab

展示正在招募或进行中的活动。顶部有本月报名人次 / 本月收入统计；卡片显示时间、地点、报名进度条、费用；可操作「查看报名」「编辑」「下架」。点卡片主体可进赛事详情或活动详情（按类型）。

### 2.3 招募已结束 Tab

展示已结束或被下架/关闭的招募。卡片样式偏灰；可「查看名单」；「数据」按钮当前提示「功能开发中」。空态无发布按钮。

## 3. 页面详细描述

### 3.1 顶部 Tab

| Tab | 说明 |
|-----|------|
| 草稿 | 未发布招募，括号内为数量 |
| 招募 | 招募中 / 进行中，括号内为数量 |
| 招募已结束 | 已结束 / 已下架，括号内为数量 |

切换 Tab 只换列表内容，页面不整体刷新。

### 3.2 月统计（草稿 Tab 不显示）

| 展示 | 说明 |
|------|------|
| 数字 + **本月报名人次** | 当前为演示数据 |
| **¥{金额}** + **本月收入** | 当前为演示数据 |

### 3.3 招募卡片

**草稿卡**

| 内容 | 说明 |
|------|------|
| 📝 + 标题 + 状态标签 | 标签随状态变化 |
| 📍 地点 | |
| 计划招募 N 人 · ¥X/人 | |
| 继续编辑 / 删除 | 删除需二次确认 |

**招募 / 招募已结束卡**

| 内容 | 说明 |
|------|------|
| 标题 + 状态标签 | 如招募中、进行中、已结束 |
| 📅 时间 · 📍 地点 | |
| 已报 X/Y 人 + 进度条 | |
| ¥X/人 | 招募已结束侧费用可弱化显示 |
| 操作按钮 | 见 §2 各 Tab 说明 |

### 3.4 空态

| Tab | 标题 | 提示 | 按钮 |
|-----|------|------|------|
| 草稿 | 暂无草稿 | 创建赛事招募后可保存为草稿，方便稍后继续编辑 | 发布赛事招募 |
| 招募 | 暂无进行中的招募 | 发布赛事招募，开始招募学员与参赛选手 | 发布赛事招募 |
| 招募已结束 | 暂无已结束的招募 | 已结束的活动会显示在这里，方便查看历史数据 | 无 |

## 4. 常见路径

- **日常管理：** 我的 → 我的招募 → 切换 Tab → 编辑 / 查看报名 / 下架  
- **从空态发布：** 招募或草稿空态 → 发布赛事招募 → [发布招募](./发布招募.md) → 成功后回到本页对应 Tab  
- **草稿续编：** 草稿 Tab → 继续编辑 → [招募编辑](./招募编辑.md) → 保存后返回  
- **看报名：** 招募 Tab → 查看报名 → [报名人员列表](./报名人员列表.md)  
- **看详情：** 点非草稿卡片 → [赛事详情](./赛事详情.md) / [活动详情](./活动详情.md)（按类型）

## 6. 规则补充与验收要点

### 6.1 已对齐（产品已确认可验收）

- 三个 Tab（草稿 / 招募 / 招募已结束）及括号内数量展示正确
- 草稿卡可「继续编辑」「删除」；删除前需二次确认
- 招募 Tab 卡可「查看报名」「编辑」「下架」；点卡片主体可进赛事详情或活动详情
- 招募已结束卡可「查看名单」
- 各 Tab 空态文案与按钮符合 §3.4
- 从发布页或编辑页返回后，列表应刷新为最新状态

### 6.2 还没做完

- 月统计（本月报名人次、本月收入）仍为演示数字，需接真实经营数据
- 招募已结束 Tab「数据」按钮尚未开放
- 「已结束」与「已下架」的展示细则待产品拍板
- 非发布者能否编辑他人招募，待权限规则确认
