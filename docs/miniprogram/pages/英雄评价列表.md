# 英雄评价列表

> 状态：草稿 · P1 · M2

---

## 1. 页面概述

| 项 | 值 |
|---|---|
| 路由 | `pages/hero-reviews/index` |
| 参数 | `hero_id`（他人）或 me（自己） |
| 入口 | 英雄详情 / 我的英雄资料 ·「查看全部」 |

## 2. 功能

- 分页评价列表
- 单项：评分、内容、时间、评价人昵称（脱敏）

## 3. API

- `GET /api/heroes/{hero_id}/reviews`
- 或 `GET /api/heroes/me/reviews`
