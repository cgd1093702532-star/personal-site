## 1. 页面业务目标

营销首页是小程序默认 Tab，用来完成获客与导流：活动曝光、权益开通、英雄推荐、课程/好物/资讯浏览。

自上而下区块：

1. Banner 主视觉 +「查看活动」
2. 金刚区：预约场地 / 活动赛事 / 精致航程 / 时尚装备
3. 航海家权益卡
4. 英雄广场（横滑）
5. 精选活动与赛事
6. 精选课程
7. 精选好物
8. 新闻动态

底部 Tab：**首页 / 英雄 / 商城 / 我的**。

## 3. 页面详细描述


| 区块      | 展示                                                                           | 交互                     |
| ------- | ---------------------------------------------------------------------------- | ---------------------- |
| Banner  | 「2023.06 前往」+ 标题「企业家杯月赛」+「06.20」+「定制式高端水上潮流体育」+ 黄钮「查看活动」                     | → 招募详情                 |
| 金刚区     | 四圆形深蓝图标 + 文案                                                                 | 本期可占位                  |
| 权益卡     | 「航海家权益卡」¥300；✓优惠购船 / 课程折扣 / 活动优先；「立即开通 →」                                    | 本期 toast/占位            |
| 英雄广场    | 头像（圆角方）+ 昵称 + 项目标签（灰底白字）+ 主荣誉（橙皇冠）+ 经验副标；三行：**赛事（黄）/ 活动（橙）/ 课程（绿）** + 文案 + › | 更多→英雄 Tab；卡头→英雄详情；行→详情 |
| 精选活动与赛事 | 大图卡：时间条（色点）+ **赛事（黄）/活动（橙）** 标签、标题、地点、价格、「立即报名」                              | → 招募详情                 |
| 精选课程    | 横滑课卡：封面、标题、价格（可「待定中」）                                                        | 本期可占位                  |
| 精选好物    | 双列商品                                                                         | 本期可占位                  |
| 新闻动态    | 左图右文 +「资讯」标签                                                                 | 本期可占位                  |

## 4. 常见路径

- 看活动：Banner / 精选活动 → 招募详情  
- 看教练：英雄广场卡片头 → 英雄详情；行 → 招募/课程详情；查看更多 → 英雄 Tab  
- 开通权益：权益卡「立即开通」
