## 1. 页面业务目标

「英雄广场」是用户通过底部 **「英雄」** Tab 进入的教练列表页。

主要解决三件事：

1. **集中展示认证教练**：方便用户浏览平台已认证的英雄教练（已认证且没有禁用的教练才会显示出来）
2. **搜索**：按教练昵称、姓名、项目名称模糊搜索，实时显示搜索结果
3. **进入教练主页**：点击教练卡片进入 [英雄详情](./英雄详情.md)，可报名赛事或课程；卡片底部赛事/活动/课程行可直达对应详情

> 本页教练数据与营销首页「英雄广场」区块保持一致（同一份列表数据源）。  
> 标题注明「使用装修搭建」：列表人选与展示顺序预期由后台**装修配置**下发；第一期前端先按「已认证且启用」全量拉取，装修排序接口就绪后再对齐。

## 2. 登录和身份描述


| 身份          | 用户大概情况  | 页面上看到什么                          |
| ----------- | ------- | -------------------------------- |
| 全部登录/访客用户   | 公开访问    | 完整搜索栏 + 教练列表                     |
| 尚未通过认证的教练本人 | 本人也在列表中 | 1、与其他教练相同的卡片样式 2、列表仅含已认证且启用状态的教练 |




### 2.1 列表有数据

1. 正常展示教练卡片列表（已认证英雄且没有禁用教练身份）
2. 列表展示顺序读后台可配置（见 §6.2；当前临时规则见下）
3. **无赛事且无课程的英雄排在最下面**（有赛事/课程的优先展示）
4. 支持下拉刷新数据



### 2.2 列表无数据（空态）


| 场景             | 空态文案侧重                          |
| -------------- | ------------------------------- |
| 有搜索词，但一个教练都找不到 | 提示「**未找到相关教练和项目**」；副文案「试试调整搜索词」 |
| 列表加载失败 / 广场无人  | 「广场暂无数据」（图标 + 文案）               |


搜索无结果预览广场暂无数据预览

### 3.1 搜索区


| 展示内容   | 说明                                |

| 搜索输入框  | 占位：「搜索教练昵称、姓名或项目名称」；输入后实时查询       |

### 3.3 教练列表区


| 展示内容            | 说明                                                                                                                                                                      |
| --------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 结果提示            | 1、有关键词且列表非空时：「找到 N 位教练」 2、有关键词且列表空时：显示搜索无结果页面，不显示找到 0 位教练                                                                                                               |
| 教练卡片            | 1、浅蓝渐变圆角卡片 2、头像 3、昵称 4、项目标签：显示全部已设置标签（后台最多 3 个）；单标签超过 6 个字以省略号显示 5、等级标签：无图标则前台不显示；标签名称完整显示；最多显示 2 个 6、底部最多三行白底圆角条：**赛事（如：黄）/ 活动（如：橙）/ 课程（如：绿）**；有则展示，文案为「状态｜标题」（课程可仅标题） |
| 点击卡片主体          | → 英雄详情                                                                                                                                                                  |
| 点击卡片底部赛事/活动/课程行 | → 对应招募、或活动详情、或课程详情                                                                                                                                                      |

## 4. 常见路径

- **Tab 进入：** 底部「英雄」→ 拉取已认证且启用的教练列表 → 浏览/搜索
- **搜索教练：** 输入关键词 → 实时刷新列表；或点键盘「搜索」立刻查
- **清空搜索：** 点 × → 收起键盘（若有）→ 恢复全量列表并重新显示 banner
- **下拉刷新：** 下拉 → 重新拉取当前关键词下的列表
- **看教练详情：** 点击教练卡片 → 英雄详情
- **直达赛事/活动/课程：** 点击卡片底部对应行 → 招募详情 / 课程详情
- **从详情返回：** 页面再次显示时自动重新拉列表

```mermaid
stateDiagram-v2
  [*] --> 默认列表: 首次进入页面
  默认列表 --> 搜索结果列表: 输入关键词（实时）
  搜索结果列表 --> 默认列表: 清空关键词
  默认列表 --> 默认列表: 下拉刷新
  搜索结果列表 --> 搜索结果列表: 下拉刷新
  默认列表 --> 英雄详情: 点击教练卡片
  搜索结果列表 --> 英雄详情: 点击教练卡片
```
