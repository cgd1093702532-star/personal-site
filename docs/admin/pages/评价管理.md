# 评价管理（管理后台）

> 状态：草稿 · P1 · M2

---

## 1. 功能

- **Tab：** 全部 / 正常 / 已隐藏 / 已删除
- **筛选：** 场景、评分范围
- **搜索：** 内容关键词
- **表格：** 评价人、英雄、场景、评分、内容、时间、状态
- **1–2 星** 高亮「待审核」
- **操作：** 查看、隐藏、删除

## 2. API

- `GET /api/admin/reviews`
- `POST /api/admin/reviews/{review_id}/hide`
- `POST /api/admin/reviews/{review_id}/delete`
