# 报名管理（管理后台）

> 状态：草稿 · P1 · M2

---

## 1. 功能

- **Tab：** 全部 / 待确认 / 已确认 / 已取消 / 已退款
- **筛选：** 场景、支付状态、支付时间
- **搜索：** 姓名、手机号
- **表格：** 报名人、招募项目、场景、时间、支付状态、报名状态
- **操作：** 查看详情、取消报名

## 2. API

- `GET /api/admin/signups?status=&pay_status=&page=&page_size=`
- `POST /api/admin/signups/{signup_id}/cancel`
