## 1. 页面业务目标

「我的课程」是**已认证英雄**查看自己名下课程的地方。

主要解决两件事：

1. **按状态浏览**：招生进行中 / 招生已结束两个 Tab，看清哪些课还在招生、哪些已结束
2. **快速掌握报名情况**：招生进行中的课程卡片展示已报名进度与单价

当前版本可点卡片进入 [课程详情](./课程详情.md)。英雄也可从个人中心「发布招募/课程」→「申请课程」进入 [发布课程](./发布课程.md) 新增课程。

## 2. 登录和身份描述

| 身份 | 用户大概情况 | 页面上多出来 / 不一样的地方 |
|------|--------------|------------------------------|
| 已认证英雄 | 个人中心「我的课程」 | 双 Tab 课程列表 |
| 未认证 / 普通用户 | 无入口 | 不应进入本页 |

> 各 Tab 所见结构一致，仅列表数据与空态文案不同。

### 2.1 招生进行中

展示当前仍在招生或进行中的课程。卡片含时间地点、¥X/人、已报名进度条。

### 2.2 招生已结束

展示已结束的课程。卡片带「已结束」标签；无进度条（或弱化展示）。

## 3. 页面详细描述

### 3.1 顶部 Tab

招生进行中 / 招生已结束，括号内为数量。

### 3.2 课程卡片

| 元素 | 说明 |
|------|------|
| 课程名称 | 主标题 |
| 时间 · 地点 | 单行摘要 |
| ¥X/人 | 单价 |
| 已报名 X/Y + 进度条 | 仅招生进行中 Tab |
| 已报名成员 | 按钮；进入该课成员列表 |

点卡片主体 → [课程详情](./课程详情.md)。点「已报名成员」→ [报名人员列表](./报名人员列表.md)（课程成员模式）。

### 3.3 空态

| Tab | 图标 | 标题 | 提示 |
|-----|------|------|------|
| 招生进行中 | 📚 | 暂无进行中的课程 | 可从个人中心「申请课程」发布 |
| 招生已结束 | ✅ | 暂无已结束的课程 | 已结束的课程会显示在这里，方便查看历史记录 |

> 空态无页内发布按钮；发布入口在个人中心「发布招募/课程」底部弹层。

## 4. 常见路径

- **查看课程：** 我的 → 我的课程 → 切换 Tab 浏览  
- **看详情：** 点课程卡片 → [课程详情](./课程详情.md)  
- **看成员：** 点「已报名成员」→ [报名人员列表](./报名人员列表.md)  
- **前台申请：** [个人中心](./个人中心.md) → 发布招募/课程 → 申请课程 → [发布课程](./发布课程.md) → 回到本页  
- **后台维护后回看：** 管理后台发布/编辑课程 → 本页刷新可见

## 6. 规则补充与验收要点

### 6.1 已对齐（产品已确认可验收）

- 两个 Tab（招生进行中 / 招生已结束）及括号内数量展示正确
- 卡片展示课程名称、时间地点、单价；招生进行中 Tab 有报名进度条
- 招生已结束 Tab 卡片带「已结束」标签，进度条弱化或不展示
- 空态文案符合 §3.3；发布入口在个人中心弹层
- 点击卡片进入课程详情
- 「已报名成员」进入成员列表（报名时间倒序：头像、昵称、手机号、时间）
- 仅展示当前英雄自己名下的课程

### 6.2 还没做完

- 课程下架 / 手动结束流程
- 与学员名册的关联跳转
- 是否需要「草稿」Tab，待产品确认
