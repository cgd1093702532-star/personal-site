## 1. 页面业务目标

「英雄评价列表」展示**教练收到的学员评价**——对应个人中心「评分」统计的明细页。

主要解决一件事：

- **看清口碑**：谁在什么时间给了几分、写了什么内容

与 [我的评价](./我的评价.md) 区分：本页是**英雄视角（收到的评价）**；「我的评价」是**普通用户视角**。当前均为只读，无回复 / 举报。

---

## 2. 登录和身份描述

| 身份 | 用户大概情况 | 页面上多出来 / 不一样的地方 |
|------|--------------|------------------------------|
| 已认证英雄 | 个人中心点「评分」 | 评价列表；空态「暂无评分」 |
| 未认证 / 普通用户 | 无入口 | 不应进入 |

> 有评价与无评价仅差列表 / 空态；导航栏标题为 **我的评分**（与文档文件名「英雄评价列表」并存，见 §6）。

---

## 3. 页面详细描述

导航栏标题：**我的评分**。

### 3.1 评价卡片

每条评价包含：

| 展示 | 说明 |
|------|------|
| 头像 | 评价者头像或占位 |
| 昵称 | 如「学员A」 |
| 星级 + 分数 | ★ 支持半星；旁侧数字分 |
| 评价正文 | 有则展示 |
| 时间 | 如 2026-06-08 |

按评价时间从新到旧排列。点击卡片当前无反应。

### 3.2 空态

| 文案 |
|------|
| **暂无评分** |

## 6. 规则补充与验收要点

### 6.1 已对齐（产品已确认可验收）

- 展示教练收到的评价：头像、昵称、星级（含半星）、分数、正文、时间
- 按评价时间降序排列
- 无评价时展示「暂无评分」
- 仅已认证英雄可从个人中心「评分」进入
- 仅展示平台允许对外可见的评价（与后台审核一致）
- 当前版本只读，点击卡片无反应

### 6.2 还没做完

- 导航标题「我的评分」与文档名「英雄评价列表」是否统一，待产品确认
- 评价回复 / 举报功能
- 英雄详情页是否展示评价摘要
