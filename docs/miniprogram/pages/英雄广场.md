## 1. 页面概述

| 说明 | 认证教练列表；营销首页内「英雄广场」区块的完整列表页 |

## 2. 页面结构

1. **搜索框** — 教练姓名 / 项目类型；右侧 **筛选图标** 收纳项目、排序、年限
2. **筛选面板** — 项目（全部 / 帆船 / …）、排序（综合 / 评分高→低 / …）、年限（不限 / 1-3年 / …）
3. **教练卡片列表** — 与首页英雄广场卡片同布局：头像、姓名、项目标签、头衔徽章、赛事/课程行

### 设计规范

- 列表卡片：参考 `DESIGN-SPEC` §4.7 简化版
- 项目标签：橙色技能 tag；年限：蓝色 tag；资质：浅蓝 tag
- 间距：`--space-page-x`

## 3. 功能说明

| 操作 | 行为 |
|------|------|
| 搜索 | **模糊搜索**：姓名、项目、资质标签、简介、**工作年限**、评分；输入防抖 300ms |
| 搜索无结果 | 展示空状态：「未找到相关教练和项目」+ 提示文案 |
| 搜索清空 | 输入内容后显示一键清空按钮 |
| 搜索有结果 | 列表上方展示「找到 N 位教练」 |
| 筛选 | 点击搜索框右侧筛选图标，底部面板选择项目 / 排序 / 年限；非默认时图标高亮 |
| 点击卡片头部 | → [英雄详情](./英雄详情.md) |
| 点击「赛事」行 | → [招募详情](./招募详情.md)（从右滑入） |
| 点击「课程」行 | → [课程详情](./课程详情.md)（从右滑入） |
| 下拉 / 上拉 | 刷新 / 分页 |
