## 1. 页面概述

| 项 | 值 |
|---|---|
| 路由 | `miniprogram/pages/my-recruitments/my-recruitments` |
| 入口 | 个人中心 · 英雄中心 ·「我的招募」 |

## 2. 页面结构

1. **Tab** — 草稿(n) / 进行中(n) / 已结束(n)（草稿 Tab 置首）
2. **本月数据概览** — 本月报名人次、本月收入
3. **招募卡片** — 标题、状态标签、时间、地点、报名进度条、费用
4. **操作按钮**（按状态）
   - 进行中/报名中：**查看报名**、**编辑**、**下架**
   - 已截止/已结束：**查看名单**、**数据**
   - 草稿：**继续编辑**、**删除**（独立卡片样式，无报名进度条）

## 3. API

`GET /api/recruitments/me?status=&page=&page_size=`
