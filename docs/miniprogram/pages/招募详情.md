# 招募详情

> 状态：草稿 · P0 · M1

---

## 1. 页面概述

| 项 | 值 |
|---|---|
| 路由 | `pages/recruitment-detail/index` |
| 参数 | `recruit_id` |

## 2. 页面结构

1. **头部** — 标题、发布英雄名、封面图（单张无指示器；多张可左右滑动并显示轮播指示器）
2. **基本信息** — 活动时间、地点、费用
3. **招募进度** — 已报/总数、进度条、剩余名额
4. **活动详情** — 适合人群、流程、注意事项（富文本）
5. **已报名人员** — 头像+昵称列表
6. **底部报名按钮** — 固定底栏

## 3. 功能说明

### 报名表单（弹层 / 子页）

| 字段 | 必填 | 校验 |
|------|------|------|
| 联系人姓名 | 是 | 非空 |
| 联系电话 | 是 | 11 位手机号 |
| 备注 | 否 | — |

**提交：** `POST /api/recruitments/{recruit_id}/signup`

### 业务规则

| 条件 | 按钮状态 |
|------|----------|
| 满员 `full` | 禁用，文案「已满员」 |
| 活动已开始 | 禁用 |
| 已过 `signup_deadline` | 禁用 |
| 未登录 | 引导登录 |

## 4. API

- `GET /api/recruitments/{recruit_id}` — 详情含进度、已报人员
