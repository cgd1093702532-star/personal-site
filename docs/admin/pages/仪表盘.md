# 仪表盘（管理后台）

> 状态：草稿 · P0 · M1 · 平台：Web 管理后台

---

## 1. 页面概述

| 项 | 值 |
|---|---|
| 路由 | `/admin/dashboard` |
| 权限 | 管理员（本地不鉴权） |
| 源码 | `admin/src/pages/Dashboard.vue` |

## 2. 功能

### 数据卡片

- 认证英雄总数
- 进行中招募数
- 本月报名数
- 待审核数（含待审申请、待确认报名、待审主页变更等）
- 可见评价数

### 近期动态

- 英雄认证通过/驳回
- 新招募发布
- 新报名
- 新评价
- 每条：操作内容 + 时间；聚合后按时间倒序，列表截断

## 3. API

`GET /api/admin/dashboard`

## 4. 页面结构

1. 统计卡片区
2. 近期动态列表

## 5. 正常流程

打开仪表盘 → 请求 `GET /api/admin/dashboard` → 渲染统计卡片与近期动态。

## 6. 业务规则

1. 统计含：认证英雄、进行中赛事/招募、本月报名、待审申请、待确认报名、待审主页变更、可见评价。
2. 近期动态由多类事件聚合，按时间倒序，列表截断展示。
3. 数据以本地 API / SQLite 为准，不以 `app_state` 为正式源。

## 7. 异常与边界

- API 未启动时提示无法加载（需启动本地 API）。
- 某类统计为空时卡片显示 0，不阻断整页。
