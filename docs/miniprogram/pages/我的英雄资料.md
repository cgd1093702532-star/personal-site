## 2. 页面结构

与 [英雄详情](./英雄详情.md) 相同头部布局；下方区块支持编辑（Mock）：

1. **头部** — 封面、头像、姓名、评分、统计（只读展示）
2. **关于我** — 展示 +「编辑」→ [简介编辑](./简介编辑.md)（`field=about`）
3. **过往荣誉** — 左滑露出编辑/删除；「新增」打开表单弹层
4. **精彩瞬间** — 添加图片、删除、长按拖动排序（小程序：长按后拖到目标位置松手交换）
5. **资质证书** — 添加、删除、拖动排序

> 不含「赛事招募」区块（与详情页区分）。

## 3. API

| 接口 | 用途 |
|------|------|
| `GET /api/heroes/me` | 资料 |
| `GET /api/heroes/me/recruitments` | 招募列表 |
