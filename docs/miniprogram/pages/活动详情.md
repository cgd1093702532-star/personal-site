## 1. 页面业务目标

「活动详情」展示某场 **活动** 的完整信息，并支持用户报名或已认证教练发起招募。与 [赛事详情](./赛事详情.md) 共用同一套页面壳与交互，差异主要在类型标签与入口数据。

主要解决四件事：

1. **看清活动信息**：类型标签（活动）、时间地点、价格、活动详情与组织者介绍
2. **立即报名**：未认证用户在招募进行中可填写姓名+手机提交报名
3. **立即报名（已认证教练）**：已认证教练在招募进行中可点「立即报名」，经二次确认后进入我的活动赛事查看
4. **状态感知**：底栏主按钮随身份 × 招募状态变化

## 2. 登录和身份描述

底栏左侧固定 **首页**、**客服**；右侧主按钮按身份与招募状态显示：

| 身份 | 招募进行中 | 招募已结束 |
|------|------------|------------|
| 已认证教练 | **立即报名**（可点 → 二次确认 → 我的活动赛事） | **招募已结束**（禁用） |
| 未认证用户 | **立即报名**（可点 → 报名表单） | **活动已结束**（禁用） |
| 从「发布赛事招募」进入 | 未发起：**发起招募**；已发起：**招募中...**（禁用） | **活动已结束**（禁用） |

### 2.1 活动存在

正常展示封面、活动信息、详情区块、固定底栏。

### 2.2 活动不存在

页面提示「活动不存在」，页面不渲染。

## 3. 页面详细描述

### 3.1 自定义导航

| 展示内容 | 说明 |
|----------|------|
| 返回按钮 | ‹ → 返回上一页 |
| 滚动标题 | 页面向下滚动后显示活动标题（沉浸导航） |

### 3.2 封面区

封面多图时轮播并自动切换；**仅一张封面时不轮播**，静态展示。

### 3.3 活动信息

页面主体为多张信息卡片：

| 卡片 | 内容 |
|------|------|
| 概要卡 | **活动**类型标签、标题、内容标签（最多 3 个）、价格 + 分享 |
| 分享 | 1、点击分享打开底部操作面板（交互对齐 [英雄详情](./英雄详情.md)）<br>2、第一组：「分享好友」「海报」<br>3、第二组：「取消」；点遮罩或取消关闭<br>4、海报：封面图 + 活动标题 + 简介摘录 + 识别码；可「分享给好友」「保存海报」 |
| VIP 引导条 | 「VIP会员卡 · 可享5大权益」+「立即尊享 ›」（本期 toast「即将开放」） |
| 时间地点卡 | 时间、地点（带 ›）、主办（建筑图标 +「{教练名}主办」）、备注（标题在上、内容在下） |
| 活动详情卡 | 「活动详情」描述 + 组织者多维介绍 |

### 3.4 固定底栏

- 左侧：**首页**（回营销首页）、**客服**（第一期提示「功能开发中」）
- 右侧：主按钮文案与禁用态见 §2（可点 / 禁用；不展示价格）
- 自 [营销首页](./营销首页.md) 英雄广场横滑（`from=home`）或 [英雄详情](./英雄详情.md) 活动与赛事（`from=hero`）进入时：点「立即报名」本期暂不响应（无弹窗、无提示）；按钮仍按 §2 展示

### 3.5 报名表单弹窗

标题「填写报名信息」：

| 字段 | 说明 |
|------|------|
| 联系人姓名 | 必填 |
| 联系电话 | 必填，11 位手机号 |
| 备注 | 选填 |
| 取消 / 提交 | 关闭弹窗 / 提交报名 |

**校验失败时的页面提示：**

| 情况 | 提示原文 |
|------|----------|
| 姓名为空 | 「请填写联系人」 |
| 手机号格式不对 | 「手机号格式不正确」 |
| 已报名再点 | 「您已报名」 |
| 提交成功 | 「报名成功」 |
| 提交失败 | 「报名失败」 |

### 3.6 发起招募二次确认

已认证教练点「立即报名」时，当前页弹窗（与赛事详情共用同一套文案）：

| 元素 | 文案 |
|------|------|
| 标题 | 确认发起赛事招募 |
| 正文 | 确认发起赛事招募后，在我的页>服务中心>我的活动赛事中查看。 |
| 取消 | 关闭弹窗 |
| 确认开始招募 | 关闭弹窗，并在**当前页底部滑出**成功提示 |

### 3.7 发起成功底部弹层

| 元素 | 文案 / 行为 |
|------|------|
| 正文 | 已成功发起招募，快去看看吧，邀请更多队友的加入 |
| 去查看 | 进入 [我的活动赛事](./我的招募.md) |
| 取消 | 关闭弹层，留在本页 |

## 4. 常见路径

- **浏览活动：** 营销首页 / 英雄广场 / 英雄详情 → 进入本页 → 展示详情
- **报名（未认证）：** 点「立即报名」→ 填表单 → 提交成功
- **立即报名（已认证教练）：** 点「立即报名」→ 二次确认 → 底部成功提示（取消 / 去查看）
- **从发布赛事招募进入：** 未发起显示「发起招募」；已发起显示「招募中...」（不可点）
- **招募/活动已结束：** 主按钮不可点
- **首页 / 客服：** 底栏左侧入口
- **返回：** ‹ 或系统返回 → 上一页

```mermaid
stateDiagram-v2
  [*] --> 用户进行中: 未认证且招募进行中
  [*] --> 教练进行中: 已认证且招募进行中
  [*] --> 用户已结束: 未认证且招募已结束
  [*] --> 教练已结束: 已认证且招募已结束
  用户进行中 --> [*]: 立即报名
  教练进行中 --> [*]: 立即报名
  用户已结束 --> [*]: 活动已结束(禁用)
  教练已结束 --> [*]: 招募已结束(禁用)
```
