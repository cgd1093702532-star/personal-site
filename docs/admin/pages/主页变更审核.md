# 主页变更审核（管理后台）

> 状态：草稿 · P2 · M3

---

## 1. 功能

- **Tab：** 待审核 / 已通过 / 已驳回
- **列表：** 英雄、变更类型、提交时间
- **对比视图：** 修改前 / 修改后，差异高亮
- **预警：** 异常变更（如大面积删除荣誉）
- **操作：** 通过、驳回、要求补充材料

## 2. API

- `GET /api/admin/profile-changes?status=&page=&page_size=`
- `GET /api/admin/profile-changes/{change_id}`
- `POST .../approve` · `POST .../reject`
