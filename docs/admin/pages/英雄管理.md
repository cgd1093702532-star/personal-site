# 英雄管理（管理后台）

> 状态：草稿 · P0 · M1

---

## 1. 页面概述

| 项 | 值 |
|---|---|
| 路由 | `/admin/heroes` · `/admin/heroes/:id` |

## 2. 英雄列表

- **Tab：** 待审核 / 已通过 / 已驳回
- **筛选：** 场景、申请时间范围
- **搜索：** 姓名、手机号
- **表格：** 头像、姓名、手机号、项目类型、申请时间、状态
- **操作：** 查看、批准、驳回

## 3. 英雄详情审核

- 基本信息、数据统计、简介
- 过往荣誉（含认证状态）
- 资质证书（点击放大）
- **批准：** `POST /api/admin/heroes/{hero_id}/approve`
- **驳回：** `POST /api/admin/heroes/{hero_id}/reject`（填原因）

## 4. API

- `GET /api/admin/heroes?status=&page=&page_size=`
- `GET /api/admin/heroes/{hero_id}`
