# 招募列表（管理后台）

> 状态：已实现 · P0 · M1 · 平台：Web 管理后台  
> 预览：`preview/admin/recruitments.html`  
> 最后更新：2026-07-13

---

## 1. 页面概述

| 项 | 值 |
|---|---|
| 页面名称 | 招募列表 |
| 面包屑 | 供需管理 / 招募列表 |
| UI 基准 | 与 [供方列表](./供方列表.md) 同款：筛选面板 + 操作栏 + Tab + 表格卡片 |

---

## 2. 筛选区字段

| 字段 | 控件 |
|------|------|
| 供方id | 文本输入 |
| 供方昵称/手机号码 | 文本输入 |
| 招募名称 | 文本输入 |
| 招募类型 | 下拉（活动 / 课程 / 赛事） |
| 招募项目 | 下拉（帆船 / 桨板 / 潜水 / 冲浪 / 皮划艇 / 游艇） |
| 招募状态 | 下拉（招募中 / 报名中 / 已满员 / 已结束 / 已取消） |

操作按钮：查询 / 重置 / 导出 / 查看已生成订单报表

---

## 3. 操作栏与 Tab

- **创建供方招募**（主按钮，M1 占位）
- Tab：**供方招募** | **平台招募**

---

## 4. 表格列

| 列 |
|----|
| 招募id |
| 供方id |
| 供方信息（头像 + 昵称 + 手机） |
| 招募名称 |
| 招募类型 |
| 招募项目 |
| 招募人数（如 `1/5`） |
| 招募状态 |
| 招募周期 |
| 创建时间 |
| 操作（详情 / 停止招募 / 继续招募） |

---

## 5. API

| 接口 | 用途 |
|------|------|
| `GET /api/admin/recruitments` | 招募列表 |
| `GET /api/admin/recruitments/{recruit_id}` | 招募详情 |
| `GET /api/admin/recruitments/{recruit_id}/signups` | 报名人员 |
| `POST /api/admin/recruitments/{recruit_id}/close` | 停止招募 → `cancelled` |
