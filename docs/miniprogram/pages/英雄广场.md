## 说明

1. 使用装修页面搭建
2. 此页面比上一期增加了搜索和列表排序规则等。

## 1. 页面业务目标

1. **集中展示认证教练**：方便用户浏览平台已认证的英雄教练（已认证且没有禁用的教练才会显示出来）
2. **搜索**：按教练昵称、姓名、项目名称模糊搜索，实时显示搜索结果
3. **进入教练主页**：点击教练卡片进入 [英雄详情](./英雄详情.md)，可报名赛事、活动、课程；卡片底部赛事/活动/课程行可直达对应详情

## 4. 常见路径

- **Tab 进入：** 底部「英雄」→ 拉取已认证且启用的教练列表 → 浏览/搜索
- **搜索教练：** 输入关键词 → 实时刷新列表；或点键盘「搜索」立刻查
- **清空搜索：** 点 × → 收起键盘（若有）→ 恢复全量列表并重新显示 banner
- **下拉刷新：** 下拉 → 重新拉取当前关键词下的列表
- **看教练详情：** 点击教练卡片 → 英雄详情
- **直达赛事/活动/课程：** 点击卡片底部对应行 → 赛事详情 / 活动详情 / 课程详情
- **从详情返回：** 页面再次显示时自动重新拉列表

```mermaid
stateDiagram-v2
  [*] --> 默认列表: 首次进入页面
  默认列表 --> 搜索结果列表: 输入关键词（实时）
  搜索结果列表 --> 默认列表: 清空关键词
  默认列表 --> 默认列表: 下拉刷新
  搜索结果列表 --> 搜索结果列表: 下拉刷新
  默认列表 --> 英雄详情: 点击教练卡片
  搜索结果列表 --> 英雄详情: 点击教练卡片
```
