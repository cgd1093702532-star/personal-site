# 用户管理（管理后台）

> 状态：草稿 · P1 · M2 · 平台：Web 管理后台

---

## 1. 功能

- **筛选：** 注册时间
- **搜索：** 昵称、手机号
- **表格：** 用户信息、注册时间、关联英雄、报名次数
- **操作：** 查看详情、禁用/启用
- **状态：** `active` / `disabled`

## 2. 页面结构

1. 筛选 + 搜索
2. 用户列表（含报名次数聚合）
3. 禁用 / 启用

## 3. API

| 接口 | 用途 |
|------|------|
| `GET /api/admin/users?page=&page_size=` | 用户列表 |
| `POST /api/admin/users/{id}/disable` | 禁用 → `disabled` |
| `POST /api/admin/users/{id}/enable` | 启用 → `active` |

## 4. 说明

本地管理端不鉴权；不做真实微信登录。

## 5. 正常流程

打开用户列表 → 搜昵称/手机 → 查看详情或禁用/启用。

## 6. 业务规则

1. 用户状态为 `active` / `disabled`。
2. 支持按昵称、手机号搜索。
3. 报名次数由报名数据聚合展示。
4. 支持禁用与启用操作。

## 7. 异常与边界

- 对已禁用用户再次禁用、对已启用用户再次启用应幂等或提示。
- 列表空态不报错。
