# 报名管理（管理后台）

> 状态：草稿 · P1 · M2 · 平台：Web 管理后台

---

## 1. 功能

- **Tab：** 全部 / 待确认 / 已确认 / 已取消 / 已退款
- **筛选：** 场景、支付状态、支付时间
- **搜索：** 姓名、手机号
- **表格：** 报名人、招募项目、场景、时间、支付状态、报名状态
- **操作：** 查看详情、取消报名

## 2. 页面结构

1. Tab + 支付筛选 + 搜索
2. 报名列表
3. 详情 / 取消操作

## 3. API

| 接口 | 用途 |
|------|------|
| `GET /api/admin/signups?status=&pay_status=&page=&page_size=` | 报名列表（唯一正式读源） |
| `GET /api/admin/signups/{signup_id}` | 报名详情 |
| `POST /api/admin/signups/{signup_id}/cancel` | 取消报名 → `cancelled` |

## 4. 说明

报名不以 `app_state` 为正式数据源；前后台同一 SQLite，经 API 读写。

## 5. 正常流程

打开报名列表 → 按 Tab/支付/关键词筛选 → 查看详情或取消报名。

## 6. 业务规则

1. 后台唯一正式读取路径为 `signups` API / 表。
2. Tab：全部 / 待确认 / 已确认 / 已取消 / 已退款。
3. 支持按支付状态筛选；支持搜姓名、手机号。
4. 取消操作将报名状态置为 `cancelled`。

## 7. 异常与边界

- 已取消记录再次取消应提示或幂等。
- 列表为空时展示空态，不报错。
