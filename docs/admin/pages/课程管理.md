# 课程管理（管理后台）

> 状态：草稿 · P1 · M1 · 平台：Web 管理后台

---

## 1. 页面概述

| 项 | 值 |
|---|---|
| 路由 | `/admin/courses` |
| 权限 | 管理员（本地不鉴权） |
| 源码 | `admin/src/pages/Courses.vue` |

## 2. 页面结构

1. 课程列表（名称、价格、人数、授课英雄）
2. 编辑详情弹层（富文本 `detail_html`）

## 3. 功能说明

### 3.1 列表

- 展示前台已发布课程
- 操作：编辑详情

### 3.2 编辑

- 维护课程详情 HTML（`detail_html`）
- 保存后写回 API

## 4. API

| 接口 | 用途 |
|------|------|
| `GET /api/courses` | 课程列表 |
| `GET /api/courses/{id}` | 课程详情 |
| `PUT /api/courses/{id}` | 更新课程（含 `detail_html`） |

## 5. 正常流程

打开课程列表 → 选择课程 → 编辑 `detail_html` → 保存。

## 6. 业务规则

1. 列表 + 编辑 `detail_html`。
2. 前台发布的课程应出现在本列表。
3. 正式数据经 `GET` / `PUT` `/api/courses`，不以 `app_state` 为源。

## 7. 异常与边界

- 无课程时展示空态。
- 保存失败时提示错误，不静默丢改。
