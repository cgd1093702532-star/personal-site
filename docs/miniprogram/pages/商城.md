# 商城

> 单页需求文档 · 微信小程序  
> 状态：草稿 · Tab「商城」· M1 占位  
> 最后更新：2026-07-03

---

## 1. 页面概述

| 项 | 值 |
|---|---|
| 页面名称 | 商城 |
| 路由 | `miniprogram/pages/mall/mall` |
| 导航位置 | Tab「商城」（Figma 定稿第三项） |
| 源码 | `miniprogram/pages/mall/` |

**说明：** M1 为 Mock 商品列表占位；完整商城能力（购物车、下单、支付）规划在 M2+。

## 2. 页面结构（M1）

1. **页头** — 标题「好物商城」、副标题
2. **商品网格** — 双列卡片（封面、标题、价格）
3. **底部提示** — 「更多商品即将上线」

## 3. 功能说明

### 3.1 数据展示

| 字段 | 说明 |
|------|------|
| title | 商品标题 |
| price | 售价（字符串，含小数） |
| cover | 封面图（M1 渐变占位） |

### 3.2 交互

| 操作 | 行为 |
|------|------|
| 点击商品 | M1 toast「商城详情（M2）」 |
| 下拉刷新 | M2 对接 API |

### 3.3 状态

- 加载中：骨架屏（M2）
- 空状态：「暂无商品」
- 错误态：重试按钮（M2）

## 4. API（M2）

`GET /api/products?page=&page_size=`

## 5. 跳转关系

| 来源 | 目标 |
|------|------|
| Tab 栏 | 本页 |
| 营销首页「好物推荐」 | 本页或商品详情（M2） |

## 6. 设计引用

- 价格色：`--color-price`
- 卡片圆角：`--radius-md`
- 详见 `docs/DESIGN-SPEC.md`
