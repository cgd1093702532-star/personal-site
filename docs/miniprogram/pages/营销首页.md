## 2. 页面结构（与 Figma slice 一致）

1. **Banner** — 背景图、航季标签、主副标题、「查看活动」CTA
2. **金刚区** — 船艇预约 / 活动赛事 / 精选课程 / 好物推荐
3. **权益卡** — 航海家权益卡
4. **英雄广场区块** — 横向滚动教练卡片 +「查看更多」→ Tab 英雄或列表页
5. **精选活动与赛事** — 纵向活动 Banner 列表
6. **精选课程** — 横向滚动课程卡片
7. **精选好物** — 双列商品
8. **新闻动态** — 图文列表
9. **Tab Bar** — 首页 / 英雄 / 商城 / 我的（框架渲染）

设计细节见 `docs/DESIGN-SPEC.md` §3–§4。

## 3. 功能说明（M1）

### 3.1 交互

| 操作 | 行为 |
|------|------|
| 金刚区入口 | 跳转对应子页或外链（待配置） |
| 英雄广场「查看更多」 | `switchTab` → 英雄 Tab |
| 活动/课程/商品卡片 | 跳转详情（M1 可部分占位） |
| 下拉刷新 | 刷新各区块数据 |

### 3.2 状态

- 加载：区块骨架屏
- 某区块无数据：隐藏或空态

## 4. 数据来源（M1 可 mock）

- Banner、活动、课程、商品、新闻：运营配置 API（**待后端定义**）
- 英雄广场区块：`GET /api/heroes?page=1&page_size=5`
