# 评价管理（管理后台）

> 状态：草稿 · P1 · M2 · 平台：Web 管理后台

---

## 1. 功能

- **Tab：** 全部 / 正常 / 已隐藏 / 已删除
- **筛选：** 场景、评分范围
- **搜索：** 内容关键词
- **表格：** 评价人、英雄、场景、评分、内容、时间、状态
- **1–2 星** 高亮「待审核」样式
- **操作：** 查看、隐藏、删除

## 2. 页面结构

1. Tab + 筛选/搜索
2. 评价列表（低分高亮）
3. 隐藏 / 删除操作

## 3. API

| 接口 | 用途 |
|------|------|
| `GET /api/admin/reviews` | 后台看全部评价 |
| `POST /api/admin/reviews/{review_id}/hide` | 隐藏 → `hidden` |
| `POST /api/admin/reviews/{review_id}/delete` | 删除 → `deleted` |

## 4. 说明

评价正式数据在 SQLite / API，不以 `app_state` 为源。

## 5. 正常流程

打开评价列表 → 筛选/搜索 → 隐藏或删除不当评价。

## 6. 业务规则

1. 后台可查看全部评价（含隐藏/删除态）。
2. 隐藏将状态置为 `hidden`。
3. 删除将状态置为 `deleted`。
4. 1–2 星评价在列表中高亮提示。

## 7. 异常与边界

- 已隐藏/已删除记录的重复操作应提示或幂等。
- 前台可见评价集合不含 `hidden` / `deleted`。
