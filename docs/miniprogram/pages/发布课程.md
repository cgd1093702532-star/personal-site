## 1. 页面业务目标

「申请课程 / 发布课程」用于已认证英雄快速发布一门课程（Banner、名称、价格、人数），发布成功后进入 [我的课程](./我的课程.md)。

主要解决三件事：

1. **上传 Banner**：支持多图；无图时占位「点击上传 Banner 图片」
2. **填写基本信息**：课程名称、价格、人数
3. **课程详情走后台**：小程序内提示前往管理后台富文本维护

---

## 2. 登录和身份描述

| 身份 | 页面上看到什么 |
|------|----------------|
| 已认证英雄 | 个人中心选「申请课程」→ 跳转后台投票调研表单 |
| 未认证 / 未登录 | 个人中心入口按身份门禁提示，不进入申请 |

---

## 3. 页面详细描述

导航栏标题：**申请课程**（与预览页一致）。

| 区块 | 说明 |
|------|------|
| Banner | 多图轮播；「+ 添加 Banner 图片」追加；空态占位可点上传 |
| 课程名称 | 必填，2–50 字 |
| 课程价格 | 必填，元/人，≥0 |
| 课程人数 | 必填，1–100 |
| 课程详情 | 文案提示：请在管理后台用富文本维护；发布后可编辑 |
| 底栏 | 「发布课程」提交 |
