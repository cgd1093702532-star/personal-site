## 2. 业务需求

### 2.1 业务目标

- 创建赛事/活动招募：基本信息、招募设置、活动详情、联系方式
- **保存草稿**：最低标题校验 → 写 draft 列表 + localStorage `recruitment-event-draft`
- **立即发布**：全量校验 → 计算 displayStatus → redirect [我的招募](./我的招募.md)?tab=

### 2.2 适用角色与权限

| 角色 | 可否进入 |
|------|----------|
| 已认证英雄 | ✅（个人中心入口守卫） |
| 其他 | ❌ |

### 2.3 核心业务规则

1. `options.type===event` → form.type=event
2. `options.draft` → 拉招募转 form；失败则 loadDraft()
3. 否则 onLoad loadDraft() 读 storage 恢复
4. 地图选点：`wx.chooseLocation` 写 location；失败 Toast **未选择地点**
5. 封面：`chooseMedia` 1 张 → form.cover
6. 招募对象多选：`AUDIENCE_OPTIONS` toggle
7. 发布成功 removeStorage draft；redirect my-recruitments?tab={listTab}

## 4. 字段与校验矩阵

> 校验函数 `recruitment-form.validateForm(form, forPublish)`

### 4.1 发布（forPublish=true）

| 字段 key | 标签 | 必填 | 规则 | 错误 Toast | API |
|----------|------|------|------|------------|-----|
| title | 招募标题 | ✅ | 2–50 字 trim | **标题 2-50 字** | title |
| location | 活动地点 | ✅ | trim 非空 | **请填写活动地点** | location |
| start_date | 开始日期 | ✅ | 非空 | **请选择开始日期** | start_at |
| start_time | 开始时间 | ❌ | 默认 09:00 | — | start_at |
| end_date | 结束日期 | ❌ | 默认同 start | — | end_at |
| end_time | 结束时间 | ❌ | 默认 17:00 | — | end_at |
| headcount | 招募人数 | ✅ | 1–100 整数 | **招募人数 1-100** | total |
| fee | 费用 | ✅ | 0–99999 | **请填写有效费用** | fee |
| deadline | 报名截止 | ✅ | 日期 | **请选择报名截止** | deadline |
| deadline vs start | — | ✅ | deadline < start_date | **截止须早于开始日期** | — |
| desc | 详细描述 | ✅ | ≥10 字 | **详细描述至少 10 字** | description |
| phone | 咨询手机 | ✅ | `/^1\d{10}$/` | **请填写有效手机号** | phone |
| highlights | 活动亮点 | ❌ | ≤500 | — | highlights |
| audience | 招募对象 | ❌ | 多选数组 | — | audience[] |
| cover | 封面 | ❌ | 图片 path | — | cover_images[0] |

### 4.2 草稿（forPublish=false）

| 规则 | 错误 |
|------|------|
| title trim 非空 | **请至少填写标题** |

### 4.3 提交 payload 示例

```json
{
  "recruit_id": "r1700000000",
  "hero_id": "1",
  "hero_name": "小哥",
  "type": "event",
  "title": "周末帆船体验营",
  "start_at": "2026-08-01T09:00:00",
  "end_at": "2026-08-01T17:00:00",
  "location": "三亚帆船港",
  "fee": 299,
  "total": 20,
  "signed": 0,
  "description": "详细描述…",
  "deadline": "2026-07-28",
  "phone": "13800138000",
  "audience": ["成人", "新手"],
  "displayStatus": "recruiting",
  "listTab": "active"
}
```

### 4.4 发布状态 resolvePublishedStatus

| 条件 | displayStatus | listTab |
|------|---------------|---------|
| now > end_at | ended | ended |
| now ≥ start_at | ongoing | active |
| 否则 | recruiting | active |
